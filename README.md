# Tabla periodica interactiva

Instrumento HTML responsivo para estudiar la tabla periodica internacional.

Incluye busqueda, filtros, fichas de elementos, lectura por voz, respuesta por voz cuando el navegador lo permite y practica con items sobre numero atomico, electrones y neutrones.
