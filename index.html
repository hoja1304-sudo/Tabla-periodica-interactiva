Cómo extraer información

  1. Localiza el número atómico para identificar el elemento.
  2. Observa el grupo vertical para inferir propiedades semejantes.
  3. Usa el periodo horizontal para ubicar el nivel de energía principal.
  4. Compara colores para reconocer familias químicas.
  5. Relaciona símbolo, nombre y uso con los ejercicios.
